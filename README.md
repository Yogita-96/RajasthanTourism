# Rajasthan Tourism

A multi-page tourism website built with HTML, CSS, and JavaScript, showcasing four pillars of Rajasthan tourism: Heritage, Wildlife, Festivals, and Hotels. Designed with distinct backgrounds and textures for each section to reflect the colors and culture of Rajasthan.

🔗 **Live site:** [rajasthan-tourism-main.vercel.app](https://rajasthan-tourism-main.vercel.app)

---

## About This Project

This was originally built as a team project at Vanier College (Winter 2022) by Simranjeet Kaur, Nancy Bansal, and Yogita, fulfilling a web development assignment with specific technical requirements (see below). The site has since been cleaned up and redeployed.

## Features

- Four themed content pages — Heritage, Wildlife, Festivals, Hotels
- Tabbed contact page with address, call, email, and chat sections
- Email validation via JavaScript
- Responsive flexbox layout
- Custom favicon and semantic HTML5 structure
- Four dedicated stylesheets for layout, menus, forms, and tabs

## Tech Stack

- HTML5 (semantic elements)
- CSS3 (Flexbox, custom properties, hover states)
- Vanilla JavaScript (tab logic, form validation)
- Deployed on Vercel

## Getting Started

```bash
git clone https://github.com/Yogita-96/RajasthanTourism.git
cd RajasthanTourism/Rajasthan-Tourism-main
```

Open `index.html` directly in your browser, or serve it locally with any static file server.

## Original Assignment Requirements

This project was completed to satisfy the following course criteria:

#### 1. Tables, Graphics & Forms
- Tables for data and/or images
- Graphic title or logo on every page
- Footer on every page showing member name, ID, and copyright/year
- At least two forms on the contact page (email and chat)

#### 2. Menu Bars
- Horizontal and/or vertical menu bar with a home link as the first option
- Icons used alongside text in menu options

#### 3. Layout & Positioning
- HTML5 semantic elements (divs/spans used sparingly)
- Fluid Flexbox layout

#### 4. Head Section
- At least three meta tags (keywords, description, UTF charset, etc.)
- SEO-friendly tags
- Unique titles per page

#### 5. Content
- Paragraph/heading text and images on each page
- At least one external link opening in a new tab
- CSS hover rollover effects
- Footer credits with clickable student names linking to their remote server pages

#### 6. Styling
Four external stylesheets:
- `pagestyle.css` — layout and structure across all pages
- `menu.css` — horizontal and vertical menu styling
- `form.css` — form elements and input states
- `tabs.css` — tab styling

`pagestyle.css` specifically defines font family/size, link states (link, visited, focus, hover, active), table styling, and background treatments.

#### 7. JavaScript
- Tab structure on the contact page (address, call, email, chat)
- Email tab reveals a form (email, topic, message, submit)
- Chat tab reveals a form (message, send)
- JS validation restricting submissions to `vaniercollege` email accounts

#### 8. Favicon
- Custom favicon linked in the `<head>` of every page
